# サブスリーへのラン日記

陸上未経験アラフォー市民ランナーのラン日記へようこそ！

このブログでは、フルマラソンでサブスリー（3時間切り）を目指す日々のトレーニング、シューズレビュー、ランニングに関する研究や知見を共有しています。

## 主なコンテンツ

- 🏃 **デイ日記**: 日々のトレーニング記録
- 👟 **シューズレビュー**: 実際に使用したランニングシューズのレビュー
- 📚 **ランニング関連研究**: 科学的な視点からのランニング分析
- 💪 **トレーニング方法**: 効果的なトレーニングテクニック
- 🏥 **障害予防**: ランニング障害の予防と対処法
